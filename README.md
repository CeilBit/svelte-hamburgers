# Svelte Hamburgers

Animated (or not) hamburgers for Svelte based on the popular [hamburgers.css](https://jonsuh.com/hamburgers/).

# How to use?

In your Svelte files:

```svelte
<script>
    import Hamburger from 'svelte-hamburgers';
    let open;
</script>

<Hamburger bind:open />
```
